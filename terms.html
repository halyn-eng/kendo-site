01

裝備・穿著

道衣、袴與護具的穿戴方式

道衣・袴怎麼穿
稽古着（上衣） — 左襟壓在右襟上面（跟和服穿法一樣），繫好胸前的繩子。
袴（下身） — 前面裙擺對齊肚臍高度，前繩繞到後面交叉再繞回前面打結；後面的板（後腰板）卡在腰椎位置貼平。

道衣＋袴是所有護具的底層，順序永遠是先穿這個，再穿其他護具。
護具穿戴順序：垂れ → 胴 → 手ぬぐい＋面 → 小手
1. 垂れ — 護下腹和大腿，繫在腰上，五片布垂下來的那個。
2. 胴 — 護胸，背後兩條帶子交叉綁好，胸前扣好。
3. 手ぬぐい＋面 — 先用手ぬぐい把頭包好吸汗，再戴面，面紐從後面繞、在後腦勺打蝴蝶結固定。
4. 小手 — 最後穿，因為穿上小手之後手就不靈活，沒辦法再綁前面幾樣的繩子了。
道衣＋袴 → 垂れ → 胴 → 手ぬぐい＋面 → 小手（最後）
面怎麼戴
先把手ぬぐい包好頭（蓋住耳朵上緣、額頭），戴上面，面金（鐵格柵）要正對臉部中線，面紐從面兩側繞到後腦勺，交叉繞兩圈後在後面打蝴蝶結，繩尾不要拖太長（垂在背後、不要甩到眼睛前面）。太鬆會前後晃、看不準；太緊會壓迫太陽穴，練久會頭痛，抓「戴穩但不痛」的鬆緊度。
02

基本姿勢與腳步

構え、踏み込み、足さばき、素振り

中段の構え（ちゅうだんのかまえ）
標準備戰姿勢。剣先對準對方喉嚨到左眼之間。
構えが高い
剣先偏上（對準額頭以上）→ 小手露出
正常
剣先對準喉嚨到左眼
構えが低い
剣先偏下（對準胸口）→ 面露出
踏み込み（ふみこみ）
打擊瞬間右腳用力踩地，產生爆發力。不是「跳起來踩」，是左腳蹬地推出去，右腳水平滑出落地。
気（喊聲）+ 劍（竹刀到）+ 体（右腳踩地）= 同一瞬間
審判判定一本時，踏み込みの音也是依據之一。
足さばき（あしさばき）
劍道的腳步移動。前進後退、左右、斜め，意識放在左腳蹬地，不是右腳主動移動。
素振り（すぶり）
徒手揮竹刀練習。分正面素振り、斜め素振り、仕掛け素振り等。自主練習的核心項目。
03

打擊部位

面・小手・胴，以及小幅度的打法

面・小手・胴（めん・こて・どう）
道館規定不打突き（喉嚨），所以有效攻擊只有這三個：

面（メン）— 頭頂  ｜  小手（コテ）— 右手腕  ｜  胴（ドウ）— 右脇腹
小さい面（ちいさいめん）
竹刀不高舉過頭，只抬到眼睛附近就打下去。腳是弓箭步，後腳不動（老師的講法是「坐下去」的姿勢）。

動作小、出手快，適合近距離或搶時機的時候用。重點不是揮得大力，而是短距離內把速度和準度做出來。
小さい面 動作示意
小さい胴（ちいさいどう）
同樣是小幅度的揮擊。要注意左手是握緊的——不是因為打胴就把手放掉。

打胴要「切進去」，是斜著切入的軌跡，不是橫著拍過去。
小さい胴 動作示意
04

技の種類

主動出擊與應對反擊的兩大分類

仕掛け技（しかけわざ）
你主動發動的攻擊技。包含出ばな技、連続技、追い込み技等。
応じ技（おうじわざ）
對方攻擊後反擊的技。等對方動了才發動，読み能力是關鍵。
返し技（かえしわざ）— 接住對方竹刀後反打
抜き技（ぬきわざ）— 閃開對方攻擊後打
擦り上げ技（すりあげわざ）— 撥開對方竹刀後打
打ち落とし技（うちおとしわざ）— 把對方竹刀打落後打
05

応じ技詳解

幾個實際好用的応じ技，附動作流程

返し胴（かえしどう）★ 初學首選
對方打面時，接住竹刀，往右前方斜步，順勢打胴。最自然、最好練的応じ技。
對方打面 → 右前斜步 → 接住 → 打胴
擦り上げ面（すりあげめん）
對方竹刀打下來時，用自己竹刀從下往上「擦」過去撥開，順勢打面。不是硬擋，是滑開——像雨傘側面把水撥走。時機要抓得很準，初學建議先練返し胴。
對方打來 → 竹刀由下往上擦 → 撥開 → 打面
抜き面 vs 返し面
兩個都是對方打來後打面，但方式不同：
抜き面
閃開（不接觸）→ 對方落空 → 打面。適合對方打胴時。
返し面
接住（有接觸）→ 撥開 → 打面。對方力道大時借力。
抜き面 技巧要點
小手抜き面 — 對方剣先移向你右手、準備打小手瞬間，手腕微微上抬後收，身體不用大退，讓對方打空，立刻踏み込み打面。
胴抜き面 — 對方剣先右橫移、腰要轉準備打胴時，右腳微向左前斜移半步，閃開胴的軌跡，同時舉竹刀打面。
看起こり（剣先／肩／腰的預兆）→ 微幅閃避 → 同拍踏み込み → 打面 → 殘心
核心是「先閃、再打」在同一拍完成，不是閃完停頓一下才打——停頓的話對方來得及收招防守或補第二擊。閃避幅度只要剛好讓對方竹刀擦不到即可，閃太大會拉開距離、打不到面；閃太小則容易被打中，抓「剛好落空」的分寸要靠反覆練習。
06

距離與接觸

間合い、鍔迫り合い、引き技

間合い（まあい）
遠間 — 還沒進攻擊範圍，等起こり
一足一刀 — 標準攻擊距離，所有技都可發動
近間 — 太近，進入鍔迫り合い狀態
鍔迫り合い（つばぜりあい）
雙方靠太近、竹刀交叉頂在一起的狀態。頂住要馬上決定打或退，太久審判喊「分かれ」強制重來。
鍔迫り合い → 引き面 / 引き小手 / 引き胴
引き面（ひきめん）＝ 退擊面
鍔迫り合い後退後打面。
打面 → 鍔迫り合い → 左手舉起 → 踏み込み＋打面 → 後退2～3步 → 殘心
⚠ 打的瞬間是踏み込み（向前踩），打完才後退，不是退的同時打。
07

稽古種類

練習方式與読み的基礎

掛かり稽古（かかりげいこ）
你一直衝、一直打，對方（元立ち）只接招不還手。不是練「打準」，是練氣力不斷、踏み込みの爆發力、體力與精神極限、殘心習慣化。30秒～2分鐘，累到腿軟正常。
起こり（おこり）
動作的起點、預兆。読み練習的關鍵——不是看竹刀，是看肩膀、腰、剣先的起こり。
看肩膀 → 要打之前肩會先動
看腰 → 踏み込み前腰會下沉
看剣先 → 上抬 = 準備打面
道館規定：面・小手・胴 only（無突き）
