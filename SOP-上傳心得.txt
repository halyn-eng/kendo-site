上傳劍道心得 SOP（Firebase 版）
================================

現在心得直接在網站上打字發布，不用改程式碼、不用 git push。


1. 開網站
---------
https://halyn-eng.github.io/kendo-site/notes.html


2. 登入
-------
右上角點「登入」，輸入：

Email：kherjing13@gmail.com
密碼：（你自己設定的那組）

送出後畫面會出現「新增心得」的表單。


3. 寫心得
---------
表單有兩欄：

- 日期：選當天日期
- 內容：直接打字，換行用 Enter 就好

打完點「發布心得」。


4. 確認
-------
發布後畫面會馬上更新，新的心得會自動排在最上面，
其他人開這個網址也能立刻看到（不用登入就能看）。


5. 刪除舊心得（如果需要）
--------------------------
登入狀態下，每篇心得右上角會多一個「刪除」按鈕，
點下去會跳出確認視窗，確認後就會刪除。


6. 用完記得登出
----------------
右上角點「登出」，尤其是共用電腦或別人也會用這台裝置時。


常見問題
========

登入失敗
檢查 email、密碼有沒有打錯；密碼忘記的話要到
https://console.firebase.google.com 的 kendo-blog 專案，
Authentication → Users 裡面重設。

發布心得沒反應 / 出現錯誤訊息
確認自己是登入狀態（右上角要顯示「登出」而不是「登入」），
沒登入是無法發布的（權限只開放給你）。

想改網站其他頁面（不是心得）
名詞解釋、読み練習、常見錯誤、每日練習這幾頁還是純 HTML，
要改內容一樣是編輯對應的 .html 檔案 + git push，做法沒變。
