# 姿勢與腳步

## 中段の構え（ちゅうだんのかまえ）

**姿勢** — 標準備戰姿勢。剣先對準對方喉嚨到左眼之間。

| 狀態 | 問題 |
|------|------|
| ❌ 構えが高い（剣先偏上，對準額頭以上） | 小手露出 |
| ✅ 正常（剣先對準喉嚨到左眼） | 正確 |
| ❌ 構えが低い（剣先偏下，對準胸口） | 面露出 |

---

## 踏み込み（ふみこみ）

**腳步** — 打擊瞬間右腳用力踩地，產生爆發力。不是「跳起來踩」，是左腳蹬地推出去，右腳水平滑出落地。

```
気（喊聲）+ 劍（竹刀到）+ 体（右腳踩地）= 同一瞬間
```

審判判定一本時，踏み込みの音也是依據之一。

---

## 足さばき（あしさばき）

**腳步** — 劍道的腳步移動。前進後退、左右、斜め，意識放在**左腳蹬地**，不是右腳主動移動。

---

## 素振り（すぶり）

**基本練習** — 徒手揮竹刀練習。分正面素振り、斜め素振り、仕掛け素振り等。自主練習的核心項目。
