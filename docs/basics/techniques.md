# 技の種類

## 仕掛け技（しかけわざ）

你主動發動的攻擊技。包含出ばな技、連続技、追い込み技等。

---

## 応じ技（おうじわざ）

對方攻擊後反擊的技。等對方動了才發動，読み能力是關鍵。

| 技名 | 讀音 | 說明 |
|------|------|------|
| **返し技** | かえしわざ | 接住對方竹刀後反打 |
| **抜き技** | ぬきわざ | 閃開對方攻擊後打 |
| **擦り上げ技** | すりあげわざ | 撥開對方竹刀後打 |
| **打ち落とし技** | うちおとしわざ | 把對方竹刀打落後打 |

---

## 返し胴（かえしどう）★ 初學首選

對方打面時，接住竹刀，往右前方斜步，順勢打胴。最自然、最好練的応じ技。

```
對方打面 → 右前斜步 → 接住 → 打胴
```

---

## 擦り上げ面（すりあげめん）

對方竹刀打下來時，用自己竹刀從下往上「擦」過去撥開，順勢打面。不是硬擋，是滑開——像雨傘側面把水撥走。時機要抓得很準，初學建議先練返し胴。

```
對方打來 → 竹刀由下往上擦 → 撥開 → 打面
```

---

## 抜き面 vs 返し面

兩個都是對方打來後打面，但方式不同：

| 技名 | 方式 | 適合情況 |
|------|------|------|
| **抜き面** | 閃開（不接觸）→ 對方落空 → 打面 | 適合對方打胴時 |
| **返し面** | 接住（有接觸）→ 撥開 → 打面 | 對方力道大時借力 |

---

## 稽古種類

### 掛かり稽古（かかりげいこ）

你一直衝、一直打，對方（元立ち）只接招不還手。練的不是「打準」，而是：
- 氣力不斷
- 踏み込みの爆發力
- 體力與精神極限
- 殘心習慣化

30秒～2分鐘，累到腿軟正常。沒有元立ち時可用壁打ち代替。

---

## 起こり（おこり）

動作的起點、預兆。読み練習的關鍵——不是看竹刀，是看肩膀、腰、剣先的起こり。

```
看肩膀 → 要打之前肩會先動
看腰   → 踏み込み前腰會下沉
看剣先 → 上抬 = 準備打面
```
