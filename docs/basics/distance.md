# 距離與接觸

## 間合い（まあい）

| 距離 | 說明 |
|------|------|
| **遠間** | 還沒進攻擊範圍，等起こり |
| **一足一刀** | 標準攻擊距離，所有技都可發動 |
| **近間** | 太近，進入鍔迫り合い狀態 |

---

## 鍔迫り合い（つばぜりあい）

雙方靠太近、竹刀交叉頂在一起的狀態（鍔＝護手互相推擠）。打面衝過去沒停住或雙方同時前進時發生。

頂住要馬上決定打或退，太久審判喊「分かれ」強制重來。

```
鍔迫り合い → 引き面 / 引き小手 / 引き胴
```

---

## 引き面（ひきめん）＝ 退擊面

鍔迫り合い後退後打面。

```
打面 → 鍔迫り合い → 左手舉起 → 踏み込み＋打面 → 後退2～3步 → 殘心
```

:::caution 注意
打的瞬間是踏み込み（向前踩），打完才後退，不是退的同時打。
:::
