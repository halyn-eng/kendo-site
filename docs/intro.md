# ⚔️ 劍道學習手冊

**道館規定：面・小手・胴 only（無突き）**

本手冊整理初學者最需要的劍道知識，包含術語解釋、読み練習、常見錯誤分析，以及每日自主練習結構。

| [每日練習](techniques/daily-practice) | 15分鐘結構 + 週輪替表 |

## 🎯 初學者優先練這3個

1. 對方打面 → **返し胴**
2. 對方打小手 → **抜き面**
3. 鍔迫り合い後 → **引き面**
