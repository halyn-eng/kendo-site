劍道學習手冊

道館規定：面・小手・胴 only（無突き）

基本姿勢與動作
姿勢
中段の構え（ちゅうだんのかまえ）
標準備戰姿勢。剣先對準對方喉嚨到左眼之間。
構えが高い
剣先偏上 → 小手露出
正常
剣先對準喉嚨到左眼
構えが低い
剣先偏下 → 面露出
腳步
踏み込み（ふみこみ）
打擊瞬間右腳用力踩地。不是「跳起來踩」，是左腳蹬地推出去，右腳水平滑出落地。
気（喊聲）+ 劍（竹刀到）+ 体（右腳踩地）= 同一瞬間
腳步
足さばき（あしさばき）
劍道的腳步移動。意識放在左腳蹬地，不是右腳主動移動。
基本練習
素振り（すぶり）
徒手揮竹刀練習。分正面素振り、斜め素振り、仕掛け素振り等。
打擊部位
打突部位
面・小手・胴（めん・こて・どう）
面（メン）— 頭頂　小手（コテ）— 右手腕　胴（ドウ）— 右脇腹
道館規定不打突き（喉嚨）。
技の種類
応じ技
返し胴（かえしどう）★ 初學首選
對方打面時，接住竹刀，往右前方斜步，順勢打胴。
對方打面 → 右前斜步 → 接住 → 打胴
応じ技
擦り上げ面（すりあげめん）
竹刀由下往上擦撥開，順勢打面。像雨傘側面把水撥走，不是硬擋。
對方打來 → 竹刀由下往上擦 → 撥開 → 打面
距離與接觸
距離
間合い（まあい）
遠間 — 等起こり　一足一刀 — 標準攻擊距離　近間 — 進入鍔迫り合い
引き技
引き面（ひきめん）＝ 退擊面
鍔迫り合い後退後打面。踏み込み（向前踩）打完才後退。
鍔迫り合い → 左手舉起 → 踏み込み＋打面 → 後退2～3步 → 殘心
読み
起こり（おこり）
動作的起點、預兆。看肩膀、腰、剣先，比看竹刀更早判斷意圖。
看肩膀 → 要打之前肩會先動
看腰 → 踏み込み前腰會下沉
看剣先 → 上抬 = 準備打面

踏み込み：跳起來踩
錯誤
右腳主動跳起 → 力量向上
正確
左腳蹬地 → 右腳水平滑出
踏み込み：變成兩段
錯誤
竹刀到 → 停一下 → 腳踩
正確
竹刀到 = 腳踩（同一瞬間）
引き面：退的同時打
錯誤
退後的同時揮竹刀 → 力道不足
正確
踏み込み → 打到 → 才後退
読み：停在那邊不知道怎麼反應
錯誤
等竹刀打下來才反應 → 來不及
正確
看起こり → 提前反應
初學重點：「對方一動你就動」，先讓身體有反應習慣。
