# 読み練習

## 對方主動攻擊

| 對方動作 | 起こりのサイン | 你的応じ技 |
|---------|--------------|-----------|
| 打面 | 剣先上抬、肩上升 | 返し胴 / 擦り上げ面 / 抜き胴 |
| 打小手 | 剣先靠右手、前傾 | 抜き面 / 擦り上げ面 |
| 打胴 | 剣先右橫移、腰轉 | 抜き面 / 返し面 |

---

## 對方靜止或試探

| 對方狀態 | 你的仕掛け技 |
|---------|------------|
| 剣先下沉（鬆懈） | 出ばな面 |
| 後退中 | 追い込み面 |
| 構えが高い（竹刀抬高） | 小手 |
| 構えが低い（竹刀壓低） | 面 |
| 足が止まっている | 連続技（小手面） |

---

## 鍔迫り合い後

| 狀況 | 你的選擇 |
|------|---------|
| 雙方糾纏 | 引き面 / 引き小手 / 引き胴 |
| 對方推你 | 利用反力 → 引き技 |
| 對方拉你 | 跟著進 → 面 or 小手 |
| 對方靜止 | 引き面 |

---

## 間合い（距離）對照

| 距離 | 狀態 | 適合的技 |
|------|------|---------|
| 遠間（遠） | 未進入攻擊範圍 | 等起こり → 出ばな技 |
| 一足一刀（剛好） | 標準攻擊距離 | 面・小手・胴皆可 |
| 近間（近） | 太近打不開 | 鍔迫り合い → 引き技 |

---

## 🎯 初學者優先練這3個

1. **對方打面 → 返し胴**
2. **對方打小手 → 抜き面**
3. **鍔迫り合い後 → 引き面**
