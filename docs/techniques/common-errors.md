# 常見錯誤

## 踏み込み：跳起來踩

**問題**：力量往上跑，不是往前

| | 說明 |
|---|---|
| ❌ 錯誤 | 右腳主動跳起來 → 力量向上 → 弧形軌跡 |
| ✅ 正確 | 左腳蹬地推出去 → 右腳水平滑出 → 幾乎貼地 |

意識放在「**左腳蹬地**」，不是「右腳跳起」

---

## 踏み込み：變成兩段

**問題**：手腳沒有同步

| | 說明 |
|---|---|
| ❌ 錯誤 | 竹刀到 → 停一下 → 腳踩（兩個動作） |
| ✅ 正確 | 竹刀到 = 腳踩（同一瞬間） |

先放慢速度練同步。喊聲幫助同步——聲音、劍、腳三個一起出。

---

## 引き面：退的同時打

**問題**：步驟順序搞錯

| | 說明 |
|---|---|
| ❌ 錯誤 | 退後的同時揮竹刀 → 力道不足 |
| ✅ 正確 | 踏み込み（踩）→ 打到 → 才後退2～3步 |

---

## 読み：停在那邊不知道怎麼反應

**問題**：看動作而不是看起こり

| | 說明 |
|---|---|
| ❌ 錯誤 | 等竹刀打下來才反應 → 來不及 |
| ✅ 正確 | 看肩膀/腰/剣先的起こり → 提前反應 |

:::tip 初學重點
不是「想對的技」，而是「對方一動你就動」，先讓身體有反應習慣。
:::

## 構えが高い / 低い

**問題**：剣先位置不正確造成防守漏洞

| 狀態 | 問題 |
|------|------|
| ❌ 高い（對準額頭以上） | 小手和胴完全露出 |
| ❌ 低い（對準胸口以下） | 面位置露出 |
| ✅ 正確 | 剣先對準對方喉嚨到左眼之間 |
