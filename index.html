劍道學習手冊

道館規定：面・小手・胴 only（無突き）

基本姿勢與動作
姿勢
中段の構え（ちゅうだんのかまえ）
標準備戰姿勢。剣先對準對方喉嚨到左眼之間。
構えが高い
剣先偏上（對準額頭以上）→ 小手露出
正常
剣先對準喉嚨到左眼
構えが低い
剣先偏下（對準胸口）→ 面露出
腳步
踏み込み（ふみこみ）
打擊瞬間右腳用力踩地，產生爆發力。不是「跳起來踩」，是左腳蹬地推出去，右腳水平滑出落地。
気（喊聲）+ 劍（竹刀到）+ 体（右腳踩地）= 同一瞬間
審判判定一本時，踏み込みの音也是依據之一。
腳步
足さばき（あしさばき）
劍道的腳步移動。前進後退、左右、斜め，意識放在左腳蹬地，不是右腳主動移動。
基本練習
素振り（すぶり）
徒手揮竹刀練習。分正面素振り、斜め素振り、仕掛け素振り等。自主練習的核心項目。
打擊部位
打突部位
面・小手・胴（めん・こて・どう）
面（メン）— 頭頂
小手（コテ）— 右手腕
胴（ドウ）— 右脇腹

道館規定不打突き（喉嚨），所以有效攻擊只有這三個。
技の種類
仕掛け技
仕掛け技（しかけわざ）
你主動發動的攻擊技。包含出ばな技、連続技、追い込み技等。
応じ技
応じ技（おうじわざ）
對方攻擊後反擊的技。等對方動了才發動，読み能力是關鍵。
返し技（かえしわざ）— 接住對方竹刀後反打
抜き技（ぬきわざ）— 閃開對方攻擊後打
擦り上げ技（すりあげわざ）— 撥開對方竹刀後打
打ち落とし技（うちおとしわざ）— 把對方竹刀打落後打
応じ技詳解
返し胴（かえしどう）★ 初學首選
對方打面時，接住竹刀，往右前方斜步，順勢打胴。最自然、最好練的応じ技。
對方打面 → 右前斜步 → 接住 → 打胴
応じ技詳解
擦り上げ面（すりあげめん）
對方竹刀打下來時，用自己竹刀從下往上「擦」過去撥開，順勢打面。不是硬擋，是滑開——像雨傘側面把水撥走。時機要抓得很準，初學建議先練返し胴。
對方打來 → 竹刀由下往上擦 → 撥開 → 打面
応じ技詳解
抜き面 vs 返し面
兩個都是對方打來後打面，但方式不同：
抜き面
閃開（不接觸）→ 對方落空 → 打面。適合對方打胴時。
返し面
接住（有接觸）→ 撥開 → 打面。對方力道大時借力。
距離與接觸
距離
間合い（まあい）
遠間 — 還沒進攻擊範圍，等起こり
一足一刀 — 標準攻擊距離，所有技都可發動
近間 — 太近，進入鍔迫り合い狀態
接觸
鍔迫り合い（つばぜりあい）
雙方靠太近、竹刀交叉頂在一起的狀態。頂住要馬上決定打或退，太久審判喊「分かれ」強制重來。
鍔迫り合い → 引き面 / 引き小手 / 引き胴
引き技
引き面（ひきめん）＝ 退擊面
鍔迫り合い後退後打面。
打面 → 鍔迫り合い → 左手舉起 → 踏み込み＋打面 → 後退2～3步 → 殘心
注意：打的瞬間是踏み込み（向前踩），打完才後退，不是退的同時打。
稽古種類
稽古
掛かり稽古（かかりげいこ）
你一直衝、一直打，對方（元立ち）只接招不還手。不是練「打準」，是練氣力不斷、踏み込みの爆發力、體力與精神極限、殘心習慣化。30秒～2分鐘，累到腿軟正常。
読み
起こり（おこり）
動作的起點、預兆。読み練習的關鍵——不是看竹刀，是看肩膀、腰、剣先的起こり。
看肩膀 → 要打之前肩會先動
看腰 → 踏み込み前腰會下沉
看剣先 → 上抬 = 準備打面

踏み込み：跳起來踩
力量往上跑，不是往前
錯誤
右腳主動跳起來 → 力量向上 → 弧形軌跡
正確
左腳蹬地推出去 → 右腳水平滑出 → 幾乎貼地
意識放在「左腳蹬地」，不是「右腳跳起」
踏み込み：變成兩段
手腳沒有同步
錯誤
竹刀到 → 停一下 → 腳踩（兩個動作）
正確
竹刀到 = 腳踩（同一瞬間）
先放慢速度練同步。喊聲幫助同步——聲音、劍、腳三個一起出。
引き面：退的同時打
步驟順序搞錯

正確
踏み込み（踩）→ 打到 → 才後退2～3步
読み：停在那邊不知道怎麼反應
看動作而不是看起こり

看肩膀/腰/剣先的起こり → 提前反應
